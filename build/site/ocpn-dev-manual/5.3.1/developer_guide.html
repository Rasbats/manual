Developer Guide

How to contribute to OpenCPN Code

The best way to contribute to OpenCPN code is to fork the main repository from https:github.com/OpenCPN/OpenCPN], push your changes to your fork and create a pull request. There is a short guide to the whole process at [[https:guides.github.com/activities/forking/|https://github.com/OpenCPN/OpenCPN]
